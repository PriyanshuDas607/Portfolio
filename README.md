# Personal Portfolio

A modern, interactive personal portfolio website built with React, Vite, and Framer Motion. This portfolio showcases my projects, skills, and provides a seamless way to get in touch.

## Features

- **Interactive UI**: Smooth animations and page transitions using `framer-motion`.
- **Contact Form**: Integrated with EmailJS for direct communication.
- **AI Integration**: Powered by Google GenAI for smart interactions.
- **Responsive Design**: Fully responsive layout that looks great on all devices.

## Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Animations**: Framer Motion & Canvas Confetti
- **Email Service**: EmailJS
- **AI Services**: Google GenAI

## Getting Started

### Prerequisites

- Node.js installed on your local machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/PriyanshuDas607/Portfolio.git
   ```

2. Navigate into the project directory:
   ```bash
   cd Portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Environment Variables:
   Create a `.env.local` file in the root directory and set your API key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```

6. Open your browser and visit `http://localhost:3000` (or the port Vite provides) to view the application.
